## How to run
    Desktop: 
        Certifique-se de já ter instalado o python no sistema.

        Importe as bibliotecas necessarias com:
            pip intasll -r requirements.txt

        Rode o main.py

    Android:
        instale o apk na pasta /bin do repositório.
